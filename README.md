## Running

#### Prerequisites

* Xcode 8.1 on macOS Sierra 10.12.1 (12B2657) or later

#### Clone & install

* Clone this repo `git clone git@github.com:7kfpun/FinanceMacOSReactNative.git`
* `cd FinanceMacOSReactNative`
* run `npm install`

#### Start

* Run `open macos/Finance.xcodeproj`
